# FORUSER

要望どおり、以下を今回の実装で反映しました。

- 最初に `TASKS.md` へ今回の実施タスクを追記
- そのタスクを実装まで完了
- ローカルLLM（llama.cpp）連携を導入（タグ候補生成・検索要約）
- Web UIを追加し、メモを素早く残して検索できるように実装

## 追加エンドポイント
- `POST /ai/tags` : メモ本文からタグ候補生成
- `POST /quick-memo` : 迅速メモ保存（タグ省略時はAI提案を利用）
- `GET /search?q=...&tags=...` : タグ絞り込み検索

## 補足
`MEMOMO_LLM_MODEL` 未設定時はフォールバック動作します。実機で llama.cpp を使う場合は README の環境変数を設定してください。
